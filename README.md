## 🚀 Featured Projects

### 🔍 [email-threat-analyzer](https://github.com/AhmedDAH1/email-threat-analyzer)
> Modular phishing detection engine that parses raw `.eml` files and performs multi-layer analysis across headers, URLs, attachments, and body content. Scores each email 0–100 using a weighted threat model. Integrated VirusTotal API for live lookups against 70+ AV engines. 34 unit tests, GitHub Actions CI.

### 🚨 [log_threat_detector](https://github.com/AhmedDAH1/log_threat_detector)
> SIEM-style CLI tool that ingests SSH, Apache, and syslog files to detect brute-force attacks, port scans, and suspicious activity. Features a correlation engine, AbuseIPDB threat intelligence, real-time Flask dashboard, email alerting, and SQLite persistence. 14 unit tests, GitHub Actions CI.

### 📡 [network-scanner](https://github.com/AhmedDAH1/network-scanner)
> Multi-threaded network scanner built with Scapy and raw sockets. Features host discovery, port enumeration, OS fingerprinting, CVE lookup, and a web dashboard.
